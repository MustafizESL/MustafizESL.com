Unit 68: Using "Some" and "A/An"

Introduction

Objective: By the end of this unit, learners will understand how to use "some" and "a/an" correctly in various contexts.

Overview: This unit focuses on using "some" for uncountable nouns and plural countable nouns, and "a/an" for singular countable nouns.

Explanation

Using "Some": Use "some" with uncountable nouns and plural countable nouns.

  • Example: I bought some bread.

Using "A/An": Use "a" before words that begin with a consonant sound and "an" before words that begin with a vowel sound for singular countable nouns.

  • Example: I bought a hat, an apple.

Examples

Using "Some":

  • Would you like some coffee?
  • I need some money.

Using "A/An":

  • Would you like a biscuit?
  • I read a book.

Practice Exercises

Exercise 68.1: What did you buy? Use the pictures to write sentences (I bought...)

  • 1. I bought some perfume, a hat and some shoes.
  • 2. I bought a newspaper (or a paper), some flowers (or a bunch of flowers) and a pen.
  • 3. I bought some stamps, some postcards and some bread (or a loaf of bread).
  • 4. I bought some toothpaste, some soap (or a bar of soap) and a comb.

Exercise 68.2: Write a sentence with "Would you like a...?" or "Would you like some...?"

  • 1. Would you like some cheese?
  • 2. Would you like some coffee? (or a cup of coffee?)
  • 3. Would you like a biscuit?
  • 4. Would you like some bread? (or a piece of bread / slice of bread?)
  • 5. Would you like a chocolate?
  • 6. Would you like some cake? (or a piece of cake?)

Exercise 68.3: Write "a/an" or "some"

  • 1. I read a book and listen to some music.
  • 2. I need some money. I want to buy some food.
  • 3. We met some interesting people at the party.
  • 4. I'm going to open a window to get some fresh air.
  • 5. Rachel didn't eat much for lunch - only an apple and some bread.
  • 6. We live in a big house. There's a nice garden with some beautiful trees.
  • 7. I'm going to make a table. First I need some wood.
  • 8. Listen to me carefully. I'm going to give you some advice.
  • 9. I want to write a letter. I need some paper and a pen.

Exercise 68.4: Which is right? Complete the sentences

  • 1. I'm going to buy some new shoes.
  • 2. Martin has got brown eyes.
  • 3. Paul has got short black hair.
  • 4. The tour guide gave us some information about the city.
  • 5. We're going to buy some new chairs.
  • 6. We're going to buy some new furniture.
  • 7. It's hard to find a job at the moment.
  • 8. We had wonderful weather when we were on holiday.

Summary

In this unit, we learned how to use "some" and "a/an" to refer to uncountable and countable nouns. We practiced forming sentences using these structures to enhance communication skills.
