Introduction

Objective: By the end of this unit, learners will understand how to use infinitives and prepositions to form complete sentences.

Overview: This unit focuses on completing sentences with the correct form of verbs and prepositions, using "to" or "for" where appropriate.

Explanation

Using Infinitives (to + verb): Use the base form of the verb with "to" to indicate purpose or intention.

  • Example: I went to the post office to get some stamps.

Using "for": Use "for" to indicate the purpose or reason for an action.

  • Example: I'm going to London for an interview.

Examples

Using Infinitives:

  • I turned on the television to watch the news.
  • I went for a walk to get some fresh air.

Using "for":

  • I'm going to London for an interview.
  • Do you have time for a cup of tea?
