Unit 99: Using "If" and "When"

Introduction

Objective: By the end of this unit, learners will understand how to use "if" and "when" correctly to describe conditions and time in sentences.

Overview: This unit focuses on using "if" to describe conditions and "when" to describe time, making writing clearer and more precise.

Explanation

Using "If": Use "if" to describe a condition that might happen.

  • Example: If you don't hurry, you'll be late.

Using "When": Use "when" to describe a specific time when something will happen.

  • Example: When I come back, we can have lunch.

Examples

Using "If":

  • If you don't want this magazine, I'll throw it away.
  • If you pass the exam, you'll get a certificate.
  • If you need money, I can lend you some.

Using "When":

  • When I got to the hotel, there were no rooms.
  • When I'm tired, I like to watch TV.
  • When the programme ended, I turned off the TV.

Summary

In this unit, we learned how to use "if" to describe a condition that might happen and "when" to describe a specific time when something will happen. We practiced forming sentences using these conjunctions to enhance clarity and precision in writing.
