Unit 111: Using Prepositions with Different Contexts

Introduction

Objective: By the end of this unit, learners will understand how to correctly use prepositions like "on," "at," "by," "with," "from," "about," etc., in different contexts.

Overview: This unit focuses on using these prepositions to describe various situations, making sentences clearer and more precise.

Explanation

Using "On": Use "on" with specific phrases and contexts.

  • Example: We heard the news on the radio.

Using "At," "By," "With," etc.: These prepositions are used in different contexts to describe actions, situations, or positions.

  • Example: I cut the paper with a pair of scissors.

Examples

Using "On":

  • We heard the news on the radio.
  • Please don't be late. Try to be here on time.
  • I won't be here next week. I'm going on holiday.
  • 'Did you see Linda?' 'No, but I talked to her on the phone.'
  • 'What's on television this evening?' 'Nothing that I want to watch.'

Using "At," "By," "With," etc.:

  • I cut the paper with a pair of scissors.
  • She usually goes to work by car.
  • Who is the woman with short hair?
  • They are talking about the weather.
  • The car is on fire.
  • She's listening to some music by Mozart.
  • The plane is flying at 600 miles an hour.
  • They're on holiday.
  • Do you know the man with sunglasses?
  • He's reading a book about grammar by Vera P. Bull.

Summary

In this unit, we learned how to use prepositions like "on," "at," "by," "with," "from," "about," etc., to describe various situations. We practiced forming sentences using these prepositions to enhance clarity and precision in writing.
