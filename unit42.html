Unit 42: Using "Too" and "Either", and Responses with "So" and "Neither"

Introduction

Objective: By the end of this unit, learners will understand how to use "too" and "either" for agreement and how to respond with "So..." and "Neither..." to show shared experiences or opinions.

Overview: This unit focuses on using "too" and "either" to express agreement and using "So..." and "Neither..." to respond in conversation.

Explanation

Using "Too" for Positive Agreement: Use "too" to agree with a positive statement.

  • Example: I'm happy. I'm happy too.

Using "Either" for Negative Agreement: Use "either" to agree with a negative statement.

  • Example: I'm not hungry. I'm not hungry either.

Responding with "So...": Use "So..." to show that you have the same positive experience or opinion as someone else.

  • Example: I'm thirsty. So am I.

Responding with "Neither...": Use "Neither..." to show that you have the same negative experience or opinion as someone else.

  • Example: I can't go to the party. Neither can I.

Examples

Using "Too" and "Either":

  • I'm going out. I'm going out too.
  • Jenny can't drive a car. She can't ride a bicycle either.

Responding with "So..." and "Neither...":

  • I need a holiday. So do I.
  • I don't know what to do. Neither do I.

Practice Exercises

Exercise 42.1: Write "too" or "either"

  • 1. I'm happy. I'm happy too.
  • 2. I'm not hungry. I'm not hungry either.
  • 3. I'm going out. I'm going out too.
  • 4. It rained on Saturday. It rained on Sunday too.
  • 5. Jenny can't drive a car. She can't ride a bicycle either.
  • 6. I don't like shopping. I don't like shopping either.
  • 7. Emma's mother is a teacher. Her father is a teacher too.

Exercise 42.2: Answer with "So..." (So am I / So do I / So can I etc.)

  • 1. I went to bed late last night. So did I.
  • 2. I'm thirsty. So am I.
  • 3. I've just had dinner. So have I.
  • 4. I need a holiday. So do I.
  • 5. I'll be late tomorrow. So will I.
  • 6. I was very tired this morning. So was I.

    • Answer with "Neither..." (Neither am I / Neither do I / Neither can I etc.)

  • 7. I can't go to the party. Neither can I.
  • 8. I didn't phone Alex last night. Neither did I.
  • 9. I haven't got any money. Neither have I.
  • 10. I'm not going out tomorrow. Neither am I.
  • 11. I don't know what to do. Neither do I.

Exercise 42.3: You are talking to Maria. Write sentences about yourself. Where possible, use "So ... I" or "Neither ... I"

  • 1. I'm learning English. So am I.
  • 2. I can ride a bicycle. So can I. / I can't.
  • 3. I'm not American. Neither am I. / I am.
  • 4. I like cooking. So do I. / I don't.
  • 5. I don't like cold weather. Neither do I. / I do.
  • 6. I slept well last night. So did I. / I didn't.
  • 7. I've never been to Scotland. Neither have I. / I have.
  • 8. I don't use my phone much. Neither do I. / I do.
  • 9. I'm going out tomorrow evening. So am I. / I'm not.
  • 10. I haven't got a headache. Neither have I. / I have.
  • 11. I didn't watch TV last night. Neither did I. / I did.
  • 12. I go to the cinema a lot. So do I. / I don't.

Summary

In this unit, we learned how to use "too" and "either" for agreement and how to respond with "So..." and "Neither..." to show shared experiences or opinions. We practiced forming sentences using these structures to enhance conversational skills.
