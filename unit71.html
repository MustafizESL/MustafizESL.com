Unit 71: Using Articles with Places

Home About Courses Contact

Introduction

Objective: By the end of this unit, learners will understand how to use articles (the, a/an) correctly with places.

Overview: This unit focuses on the rules for using articles with place names and locations, and how to use them in sentences appropriately.

Explanation

Using Articles with Places: Some places do not require an article, while others do. The rules depend on the type of place and the context.

Examples

No Article:

  • He's in bed.
  • She's in hospital.

With Article ("the"):

  • They're at the cinema.
  • I need to go to the bank.

Summary
