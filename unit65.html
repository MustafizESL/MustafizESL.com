Unit 65: Using "A" and "An"

Introduction

Objective: By the end of this unit, learners will understand how to use "a" and "an" correctly in various contexts.

Overview: This unit focuses on using the indefinite articles "a" and "an" to refer to singular, non-specific nouns.

Explanation

Using "A": Use "a" before words that begin with a consonant sound.

  • Example: a book, a car.

Using "An": Use "an" before words that begin with a vowel sound.

  • Example: an apple, an hour.

Examples

Using "A":

  • a window.
  • a horse.

Using "An":

  • an old book.
  • an airport.
