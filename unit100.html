Introduction

Objective: By the end of this unit, learners will understand how to form and use conditional sentences correctly in various contexts.

Overview: This unit focuses on using conditional sentences to describe hypothetical situations and their potential outcomes.

Explanation

Conditional Sentences: Conditional sentences describe a condition and its possible outcome. They often use "if" to introduce the condition.

  • Example: If I knew the answer, I'd tell you.

Examples

Using Conditionals:

  • If he had the money, he would buy a fast car.
  • If she knew, she'd be angry.

Practice Exercises

Exercise 100.1: Complete the sentences

  • 1. I don't know the answer. If I knew the answer, I'd tell you.
  • 2. I have a car. I couldn't travel very much if I didn't have a car.
  • 3. I don't want to go out. If I wanted to go out, I'd go.
  • 4. We haven't got a key. If we had a key, we could get into the house.
  • 5. I'm not hungry. I would have something to eat if I was hungry.
  • 6. Sue enjoys her work. She wouldn't do it if she didn't enjoy it.
  • 7. He can't speak any foreign languages. If he could speak a foreign language, perhaps he would get a better job.
  • 8. You don't try hard enough. If you tried harder, you would have more success.
  • 9. I have a lot to do today. If I didn't have so much to do, we could go out.

Exercise 100.2: Put the verb in the correct form

  • 1. If he had the money, he would buy a fast car. (he/have)
  • 2. Jane likes living in a city. She wouldn't be happy if she lived in the country. (she/not/be)
  • 3. If I wanted to learn Italian, I would go to Italy. (I/go)
  • 4. I haven't told Helen what happened. She'd be angry if she knew. (she/know)
  • 5. If we had a map, I could show you where I live. (we/have)
  • 6. What would you do if you won a lot of money? (you/win)
  • 7. It's not a very good hotel. I wouldn't stay there if I were you. (I/not/stay)
  • 8. If we lived nearer London, we would go there more often. (we/live)
  • 9. It’s a pity you have to go now. It would be nice if you had more time. (it/be)
  • 10. I'm not going to take the job. I'd take it if the salary were higher. (the salary/be)
  • 11. I don't know anything about cars. If the car broke down, I wouldn't know what to do. (I/not/know)
  • 12. If you could change one thing in the world, what would you change? (you/change)

Summary

In this unit, we learned how to use conditional sentences to describe hypothetical situations and their potential outcomes. We practiced forming sentences using these structures to enhance communication skills.
