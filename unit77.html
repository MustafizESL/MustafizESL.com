Unit 77: Using "No" and "Any"

Home About Courses Contact

Introduction

Objective: By the end of this unit, learners will understand how to use "no" and "any" correctly in various contexts.

Overview: This unit focuses on using "no" and "any" to express the absence of something or to refer to an unspecified quantity of something.

Explanation

Using "No": Use "no" to indicate the absence of something.

  • Example: There are no shops near here.

Using "Any": Use "any" in negative statements and questions to refer to an unspecified quantity.

  • Example: There aren't any buses today.

Examples

Using "No":

  • Carla has got no free time.
  • There is no light in this room.

Using "Any":

  • We haven’t got any money.
  • There isn't any milk in the fridge.

Practice Exercises

Exercise 77.1: Rewrite the sentences with "no" or "any"

  • Write the sentences again with "no":
    • 1. We haven't got any money. -> We've got no money.
    • 2. There aren’t any shops near here. -> There are no shops near here.
    • 3. Carla hasn't got any free time. -> Carla has got no free time.
    • 4. There isn't a light in this room. -> There is no light in this room.
  • Write the sentences again with "any":
    • 5. We've got no money. -> We haven’t got any money.
    • 6. There's no milk in the fridge. -> There isn't any milk in the fridge.
    • 7. There are no buses today. -> There aren't any buses today.
    • 8. Tom has got no brothers or sisters. -> Tom hasn't got any brothers or sisters.

Exercise 77.2: Write "no" or "any"

  • 1. There's no sugar in your coffee.
  • 2. My brother is married, but he hasn't got any children.
  • 3. Sue doesn't speak any foreign languages.
  • 4. I'm afraid there's no coffee. Would you like some tea?
  • 5. 'Look at those birds!' 'Birds? Where? I can't see any birds.'
  • 6. 'Do you know where Jessica is?' 'No, I've got no idea.'
  • Write "no," "any," or "none":
    • 7. There aren’t any pictures on the wall.
    • 8. The weather was cold, but there was no wind.
    • 9. I wanted to buy some oranges, but they didn't have any in the shop.
    • 10. Everything was correct. There were no mistakes.
    • 11. 'How much luggage have you got?' 'None.'
    • 12. 'How much luggage have you got?' 'I haven’t got any.'

Exercise 77.3: Complete the sentences. Use "any" or "no" + the words in the box

  • Box: answer - difference - friends - furniture - heating - money - problems - questions - queue
  • 1. Everything was OK. There were no problems.
  • 2. Jack and Emily would like to go on holiday, but they've got no money.
  • 3. I'm not going to answer any questions.
  • 4. He's always alone. He's got no friends.
  • 5. There is no difference between these two machines. They’re exactly the same.
  • 6. There wasn't any furniture in the room. It was completely empty.
  • 7. I tried to phone you yesterday but there was no answer.
  • 8. The house is cold because there isn't any heating.
  • 9. There was no queue outside the cinema, so we didn't have to wait to get tickets.

Exercise 77.4: Write short answers (one or two words) to these questions. Use "none" where necessary

  • Example answers:
  • 1. How many letters did you write yesterday? Two. or A lot. or None.
  • 2. How many sisters have you got? Three.
  • 3. How much coffee did you drink yesterday? Two cups.
  • 4. How many photographs have you taken today? None.
  • 5. How many legs has a snake got? None.

Summary

In this unit, we learned how to use "no" and "any" to express the absence of something or to refer to an unspecified quantity. We practiced forming sentences using these structures to enhance communication skills.
