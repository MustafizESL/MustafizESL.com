Introduction

Objective: By the end of this unit, learners will understand how to use possessive nouns and pronouns correctly in various contexts.

Overview: This unit focuses on using possessive forms to indicate ownership and relationships, and how to structure sentences using these forms.

Explanation

Using Possessive Nouns: Possessive nouns show ownership and are formed by adding an apostrophe and an "s" to a noun.

  • Example: Brian is Helen's husband.

Using Possessive Pronouns: Possessive pronouns show ownership and replace nouns.

  • Example: This car is mine.

Examples

Using Possessive Nouns:

  • Sarah is Daniel's mother.
  • Helen is Brian's wife.

Using Possessive Pronouns:

  • Whose is this? Alice's.
  • Whose books are these? Mine.
