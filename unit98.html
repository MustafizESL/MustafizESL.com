Unit 98: Using "When," "While," "Before," and "After"

Home About Courses Contact

Introduction

Objective: By the end of this unit, learners will understand how to use "when," "while," "before," and "after" correctly to connect actions and events in time.

Overview: This unit focuses on using these conjunctions to indicate the timing of actions and events, making writing clearer and more logical.

Explanation

Using "When": Use "when" to indicate the time at which something happens.

  • Example: When I went out, it was raining.

Using "While": Use "while" to indicate that two actions happen at the same time.

  • Example: While we were asleep, somebody broke into the house.

Using "Before": Use "before" to indicate that one action happens earlier than another.

  • Example: They looked both ways before they crossed the road.

Using "After": Use "after" to indicate that one action happens later than another.

  • Example: After they got married, they went to live in New Zealand.

Examples

Using "When":

  • When I'm tired, I like to watch TV.
  • When I phoned her, there was no answer.

Using "While":

  • While they were away, their house was damaged in a storm.

Using "Before":

  • They looked both ways before they crossed the road.
  • Where did they live before they came here?

Using "After":

  • After they got married, they went to live in New Zealand.

Summary

In this unit, we learned how to use "when" to indicate the time at which something happens, "while" to indicate that two actions happen at the same time, "before" to indicate that one action happens earlier than another, and "after" to indicate that one action happens later than another. We practiced forming sentences using these conjunctions to enhance clarity and logic in writing.
